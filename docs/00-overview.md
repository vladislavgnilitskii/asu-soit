# АСУ СОИТ — обзор проекта

## О системе
Сервисный центр ООО «ТехноСервис-IT» — диагностика и ремонт IT-техники
для физических и юридических лиц.

## Роли пользователей
- Администратор — приём заявок, взаимодействие с клиентами
- Инженер — диагностика, ремонт, закрытие заявок
- Кладовщик — складской учёт комплектующих
- Бухгалтер — финансовые документы, отчётность
- Руководитель — аналитика, отчёты
- Системный администратор — управление системой, аудит
